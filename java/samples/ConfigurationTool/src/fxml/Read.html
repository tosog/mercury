Read(Serial readers)

  • If you choose to Auto-read on GPI you must set the GPI pin HIGH on module to begin reading.
  • If you did not choose to use Auto-read on GPI the reader will be continuously reading and will only stop with a power off.
  • NOTE: This read display only works during initial saving of autonomous configuration. Read will not stopped on the module if disconnected from configuration tool while in autonomous mode.


Read(Elara)

  • If you choose to Trigger On Boot reader will begin reading automatically after each power on.
  • If you did not choose to use Trigger On Boot the read needs to be started using Start/Stop buttons(in CDC interface) or using CapsLock(in HID interface).
  • NOTE: While read is in progress, if any settings are changed, set or saved the tool will stop the read and then apply the settings. Read needs to be started again.
